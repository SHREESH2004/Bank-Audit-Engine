Q1: HTML Text Formatting Tags Demo

This paragraph demonstrates various HTML formatting tags. The word bold uses the <b> tag for bold text. The word italic uses the <i> tag for italic text. The word underlined uses the <u> tag for underlined text. The word strong uses the <strong> tag for semantically important text. The word emphasized uses the <em> tag for emphasized text. The word highlighted uses the <mark> tag to highlight text.
